# Component - MGO

This module contains reusable UI components that are used in this project. It includes standalone components like buttons and banners that follow our design system.

## Highlighted

- **MgoBanner**: Use to show banners.
- **MgoSnackBar**: Use instead of androidx.compose.material3.SnackBar.
- **MgoAlertDialog**: Use instead of androidx.compose.material3.AlertDialog.
- **MgoBasicTextField**: Use instead of androidx.compose.foundation.text.BasicTextField.
- **MgoButton**: Use instead of androidx.compose.material3.Button.
- **MgoCard**: Use instead of androidx.compose.material3.Card.
- **MgoCheckbox**: Use instead of androidx.compose.material3.Checkbox.
- **MgoHtmlText**: Use to show text that has HTML.
- **MgoScaffold**: Use instead of androidx.compose.material3.Scaffold.
