# Feature - Framework - Storage

This module contains classes that handle local storage.

## Highlighted

- **DefaultCacheFileStore**: Use this class to save files in the app cache directory.
- **DefaultEncryptedEncryptedFileStore**: Use this class to save encrypted files securely in the 
  app directory.
- **DataStoreKeyValueStore**: Use this class to save key-value pairs.
- **EncryptedSharedPreferencesSecureKeyValueStore**: Use this class to save encrypted key-value 
  pairs.
