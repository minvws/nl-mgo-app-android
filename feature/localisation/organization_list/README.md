# Feature - Localisation - Organization List

This module contains the screens to show searched health care providers.

## Highlighted

- **OrganizationListAutomaticSearchScreen**: Use to show the screen that displays health care 
  providers that were found based on if they have data for you.
- **OrganizationListManualScreen**: Use to show the screen that displays health care
    providers that were found based on searching via a name and city.
