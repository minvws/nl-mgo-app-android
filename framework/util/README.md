# Feature - Framework - Util

This module contains helper classes for various functionality.

## Highlighted

- **DefaultBase64Util**: Use to decode and encode base64 strings.
- **accessibilityAnnounce**: Announces a string through accessibility.
- **getStringResourceByName**: Get copy programmatically based on the key. Only use this method 
  if you want to get the copy based on a dynamic string.
- **shareFile**: Use this function to share a file to another app.
- **launchBrowser**: Launch the browser for a given url.
